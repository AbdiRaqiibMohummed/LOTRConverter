🧙‍♂️ LOTR Currency Converter (SwiftUI)

A simple currency converter inspired by The Lord of the Rings, built using SwiftUI. This app allows you to convert between fictional currencies such as Copper Pennies, Silver Pieces, and Gold Pieces using a clean and efficient enum-based design.

Features

Convert between multiple LOTR currencies
Real-time conversion logic
Clean and scalable Swift enum structure
SwiftUI-based interface
Custom images for each currency type
🛠 Requirements
Xcode (latest recommended)
iOS 15+
Swift 5+

How to Run

Clone the repository:

git clone <your-repo-url>

Open the project in Xcode:

open LOTRConverter.xcodeproj
Build and run the app on a simulator or physical device
💡 How It Works
Each currency is defined as an enum with a raw value representing its base conversion rate.
Conversion is handled by a function that calculates the equivalent value between currencies.
Supports user input as a string and safely converts it into numerical values.
